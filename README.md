# 📝 Task Manager App

A simple full-stack Task Manager application built to demonstrate core frontend and backend skills including API integration, CRUD operations, and clean code structure.

---

## 🚀 Features

- ➕ Add new tasks  
- 📋 View all tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- ⏱️ Display task creation time  
- ⚠️ Basic error handling and loading state  

---

## 🛠️ Tech Stack

**Frontend:**
- React (Functional Components + Hooks)

**Backend:**
- Node.js
- Express.js

**Other:**
- REST API
- In-memory data storage

##  Setup & Run Instructions

###  1. Clone the repository


git clone https://github.com/nawaztech24/task-manager.git

cd task-manager


---

###  2. Run Backend


cd backend


npm install


node server.js


 Server will run on:

http://localhost:5000


---

###  3. Run Frontend

Open a new terminal:


cd frontend


npm install


npm start


App will run on:

http://localhost:3000
